Iniciando no desenvolvimento web

O que é a internet?

A internet é uma rede de computadores interligadas, utilizada no mundo todo, que utiliza um conjunto próprio de protocolos chamado TCP/IP.

O que é um web Browser? Dê exemplos.

Mais conhecido como navegador, um browser é um programa que permite a visualização dos conteúdos que apresenta uma página web, como imagens, vídeos e textos. Google Chrome, Opera e Microsoft Edge são alguns exemplos.

O que é um servidor?

Um servidor é um computador ,equipado com um ou mais processadores, bancos de memória, portas de comunicação, softwares e algum sistema para armazenamento de dados, como hard disks internos ou memórias SSD, que faz parte de uma rede e presta serviços a outros dispositivos, conhecidos como clientes.

O que é HTML?

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de marcação que compõe a maior parte das páginas da internet e dos aplicativos online. Com HTML pode definir a estrutura de uma página web, incluindo textos, imagens, links e outros elementos interativos.

O que são tags HTML?

As tags HTML são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.
